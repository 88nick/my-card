# Сайт-визитка

## Контекст
Мой первый веб-проект. Я только учусь. Простой одностраничный сайт о себе.

## Стек
- Чистый HTML5 + CSS3
- БЕЗ фреймворков (React, Vue, Next.js)
- БЕЗ JavaScript (пока не нужен)
- Один файл index.html со встроенными стилями

## Правила
- Объясняй каждый блок кода как новичку
- Комментарии в HTML/CSS — на русском
- Делай маленькими шагами, после каждого жду подтверждения
- Не добавляй сторонние библиотеки без явного разрешения
- Стили — внутри <style> в самом index.html, никаких внешних файлов
